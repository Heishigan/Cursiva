# Security Policy

If you're reading this, you probably found something. Respect. We take security seriously at Cursiva. We build autonomous agents, which means we have to lock down the perimeter.

## Supported Versions

Only the latest `main` branch is actively supported with security patches. Stay up to date, or proceed at your own risk.

## Reporting a Vulnerability

Found a zero-day? Bypassed our auth? Found a way to leak CV data?
**Do not open a public issue.** We don't want script kiddies poking around while we patch it.

Send an encrypted ping (or just a regular email) to:
**hej@heishi.se**

Give us a proof of concept. If the exploit is legit, we'll patch it, ship it, and give you credit in the release notes. We try to respond within 48 hours.

Hack the planet, but do it responsibly.
